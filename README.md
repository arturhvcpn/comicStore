# comicStore

## Clone Project

### Step-by-step
```sh
$ git clone https://github.com/arturhvcpn/comicStore.git
```

###  step
```sh
$ cd comicStore
```

###  Install dependencies
```sh
$ yarn 
```
#####  or

```sh
$ npm install 
```

###  If you running in MacOs execute to linking dependencies
```sh
$ npx pod-install ios 
```
